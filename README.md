🐕 Happy Dog — Cyber Runner
Un juego de runner cyberpunk donde un perro corre por las calles de una ciudad neón, atrapa huesos y esquiva obstáculos. Sin instalación, sin antivirus, funciona en cualquier PC o Mac.

🎮 Cómo jugar
Windows

Descarga o clona este repositorio
Haz doble click en Jugar_Windows.bat
El juego abre como ventana de app (sin barras del navegador)

Mac

Descarga o clona este repositorio
Haz doble click en Jugar_Mac.command
El juego abre como ventana de app (sin barras del navegador)


⚠️ Mac — error "Apple could not verify..."
La primera vez Mac puede bloquear el archivo por seguridad. Para solucionarlo:

Abre la Terminal
Escribe xattr -d com.apple.quarantine  (con espacio al final)
Arrastra el archivo Jugar_Mac.command directo a la Terminal — completa la ruta solo
Presiona Enter
Ahora haz doble click al archivo normalmente

Esto solo es necesario la primera vez. El comando únicamente elimina la etiqueta de cuarentena que Mac pone a todo archivo descargado de internet, no modifica el juego.

Cualquier sistema (alternativa directa)
Abre el archivo HappyDog.html directamente en Chrome, Edge, Firefox o Safari.

🏆 Mecánicas

Atrapa huesos para sumar puntos
Esquiva hidrantes y pájaros — cada golpe resta una vida
Cada 20 puntos la velocidad aumenta un 15%
Cada 50 puntos recuperas una vida ❤️
Tienes 5 vidas en total — el juego es infinito hasta perderlas todas
